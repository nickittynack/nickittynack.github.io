Text Styles

Just like in Microsoft Word or Google Docs, we can make italic, bold and underlined text as well as paragraphs.

It's important to remember that each tag must have a closing tag, and that tags are usually the first letter of what you are trying to do.

  • The tag for italics is i
  • The tag for bold is b
  • The tag for underline is u
  • The tag for paragraph is p

So putting those all together

HTML

<i> I'm in italics! </i>
<b> I'm bold! </b>
<u> I'm underlined </u>
<p> I'm a seperate paragraph of text </p>

RESULT

I'm in italics! I'm bold! I'm underlined

I'm a seperate paragraph of text

Layout

There's also a fwe other tags which we want to talk about.

  • The tag for centering is center (surprisingly!)
  • The tag for a line break is br

Centering is pretty straightforward

HTML

<center>I'm centered</center>

RESULT

I'm centered

The br tag is a bit more special, we don't need a closing and opening tag.

HTML

Some text</br>Line broken text!

RESULT

Some text
Line broken text!

These are useful for making things look a bit better.

Headings

Just like a normal document you will want to have headings throughout your website. There's 6 different heading sizes, h1 is the largest while h6 is the smallest. So on a site you should use h1 before you use h6.

HTML

<h5> Small heading </h5>
<h6> Smallest heading </h6>

RESULT

Small heading
Smallest heading

So now we have some nice looking text, some formatting an headers!
